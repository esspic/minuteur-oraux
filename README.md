# Minuteur d'oraux

Appli web pour chronométrer le passage d'un candidat à l'oral : exposé puis entretien.

- Presets 5+10 / 10+30 min + durées libres
- Le compteur continue en cas de dépassement (visuel orange/rouge, **jamais de son**)
- L'entretien démarre avec pour max : temps total − temps réel d'exposé
- Temps total écoulé → écran clignote en rouge, le compteur continue
- Récapitulatif des temps réels en fin d'oral
- Gros chiffres lisibles de loin, écran maintenu allumé (Wake Lock)

**Utilisation : https://esspic.github.io/minuteur-oraux/**
